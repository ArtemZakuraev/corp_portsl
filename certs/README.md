# SSL Сертификаты для Mattermost

Эта папка предназначена для хранения SSL сертификатов, которые будут автоматически добавлены в доверенные при сборке Docker образа.

## Использование

1. Поместите файлы сертификатов (`.crt` или `.pem`) в эту папку
2. Примеры имен файлов:
   - `team.rtinform.ru.crt`
   - `mattermost.crt`
   - `ca-certificate.pem`

3. При следующей сборке Docker образа сертификаты будут автоматически установлены в систему

## Альтернативный способ

Если вы используете самоподписанный сертификат и не хотите добавлять его в систему, вы можете отключить проверку SSL в настройках Mattermost (раздел "Настройки портала" → "Сервер Mattermost").

**Внимание:** Отключение проверки SSL снижает безопасность и не рекомендуется для production окружений.

## Получение сертификата

Если вам нужен сертификат для вашего Mattermost сервера, вы можете:

1. Экспортировать сертификат из браузера при подключении к Mattermost
2. Использовать команду OpenSSL:
   ```bash
   openssl s_client -showcerts -connect team.rtinform.ru:443 </dev/null 2>/dev/null | openssl x509 -outform PEM > team.rtinform.ru.crt
   ```

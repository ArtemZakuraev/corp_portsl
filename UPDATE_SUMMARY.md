# Обновление интеграции Mattermost и настроек системы

## 📋 Основные изменения

### 1. Архитектура хранения настроек

#### Системные настройки (SystemSetting)
- **Где хранятся**: Таблица `settings_systemsetting` (ключ-значение)
- **Ключи**:
  - `site_url` - публичный адрес системы
  - `mattermost_url` - адрес сервера Mattermost
  - `mattermost_webhook_url` - URL вебхука для уведомлений
  - `mattermost_bot_token` - токен системного бота
  - `verify_ssl` - проверка SSL сертификатов (True/False)
  - `background_image_url` - URL фонового изображения

#### Персональные настройки пользователя (MattermostProfile)
- **Где хранятся**: Таблица `settings_mattermostprofile`
- **Поля**:
  - `user` - связь с пользователем Django
  - `mm_username` - имя пользователя в Mattermost
  - `mm_password` - пароль (для basic auth)
  - `mm_token` - персональный токен доступа
  - `is_active` - активна ли интеграция

### 3. Интерфейс чата (Telegram-style)

**Новый шаблон**: `templates/mattermost_integration/chat.html`

Функционал:
- Список каналов слева (sidebar)
- Область переписки справа
- Автообновление сообщений (polling каждые 5 сек)
- Отправка сообщений Enter или кнопкой
- Индикатор статуса соединения
- Проверка соединения с детальной диагностикой

### 4. API Endpoints

| Endpoint | Метод | Описание |
|----------|-------|----------|
| `/mattermost/` | GET | Основной интерфейс чата |
| `/mattermost/api/channels/` | POST | Получить список каналов пользователя |
| `/mattermost/api/messages/` | POST | Получить сообщения канала |
| `/mattermost/api/send/` | POST | Отправить сообщение |
| `/mattermost/test/` | POST | Проверка соединения |

## 🔧 Настройка

### Для администратора системы

1. Перейти в **Настройки системы** → **Mattermost**
2. Заполнить:
   - **Адрес сервера Mattermost** (например, `https://mattermost.company.com`)
   - **Webhook URL** (заполняется автоматически как `{server_url}/hooks/...`)
   - **Токен бота** (создается в Mattermost → Integrations → Bots)
   - **Проверять SSL** (снять галочку для самоподписанных сертификатов)
   - **Фоновое изображение** (для эффекта стекла)

### Для каждого пользователя

1. Перейти в **Профиль** → **Настройки Mattermost**
2. Заполнить:
   - **Имя пользователя Mattermost** (логин)
   - **Пароль** ИЛИ **Персональный токен**
   - Сохранить

Персональный токен создается в Mattermost:
- Menu → Settings → Security → Personal Access Tokens → Create New Token

## ✅ Проверка работы

1. Администратор заходит в **Настройки системы**
2. Указывает адрес Mattermost, токен бота
3. Обычный пользователь заходит в **Профиль**
4. Указывает свои логин/пароль или токен Mattermost
5. Переходит в раздел **Mattermost**
6. Видит список своих каналов
7. Может отправлять и получать сообщения

- ✅/❌ Webhook настроен
- ✅/❌ Токен бота настроен
- Вкл/Выкл SSL проверка
- ✅/❌ Данные пользователя настроены
- ✅/❌ Соединение с API
- ✅/❌ Доступность сервера
